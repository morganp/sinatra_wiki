Sinatra Wiki

A simple Wiki, written in Ruby using Sinatra, built with deployment to Heroku in mind

View the Project on GitHub morganp/sinatra_wiki

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

Usage

$ git clone git://github.com/morganp/sinatra_wiki.git
$ cd sinatra_wiki
$ gem install bundler     # If not in your global gemset
$ bundle install
$ rake db:migrate_devel (sqlite) or rake db:migrate (heroku postgres) 
$ ruby sinatra_wiki.rb

Then open a browser in http://0.0.0.0:4567/
To create a page, just type in the URL eg http://0.0.0.0:4567/monkey

Required gems (in Gemfile)

  • rubygems
  • sinatra
  • erb
  • maruku
  • thin
  • haml
  • sass
  • activesupport
  • sinatra-cache
